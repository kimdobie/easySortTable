Examples for the jquery.easySortTable.js file

You call the easySortTable method to a table object using jQuery "daisy-chaining - for example : $('#myTable').easySortTable();

Because I wanted to apply the easySortTable to every table on this page, the following code was add to the head of the page:

<script type="text/javascript">
       $(document).ready(function(){
          $('table').each( function(){$(this).easySortTable(); });
       });
  </script>

If the easySortTable is not applying the header links to the proper row, you can define the exact row you want to contain the header links. The easiest way of doing this is by wrapping the single header row in a <thead> tag. If that doesn't work, apply the "sortHeader" class to the row you want to contain the links.

For example, this is the html for the above table:

<table>
	<thead>
		<tr class="sortHeaderRow">
			<th>Name</th>
			<th>Bonus</th>
			<th> Extension</th>
			<th>Start date</th>
		</tr>
		<tr>
			<th colspan="4">Current Employees Below</th>
		</tr>
	
	</thead>

If you don't want a column to have a sort link, you can add the "noSort" class to the table cell.

Example 5 - Use of Custom Data Type

Ice Cream Flavor Rating
Vanilla Good
Chocolate Average
Strawberry Poor
Rocky Road Average
Rum Raisin Poor
Mint Chocolate Chip Good

By default, this library tries to figure out what type of data is contained in a column. If a column doesn't seem to be sorting correctly, you may need to manually define the type of data in that column. Currently there are three built-in data types:

  • dataType_alpha - Text
  • dataType_numeric - Numbers including currency (US)
  • dataType_date - Dates (mm/dd/yyyy only)

To manually apply a data type, add it as a class to the header cell.

If needed, you can define your own data types. For example, in the rating column in the above table, I want the following for sorting purposes:

  • Poor to equal 0
  • Average to equal 1
  • Good to equal 2

To do this, I need to create a custom function. This function name must start in "dataType_" , have the jQuery object for the cell as the attribute, and return a value. For example the following function was created for the Rating column:

 function dataType_rating($cell){
		if($cell.text()=="Good") return 2;
		if($cell.text()=="Average") return 1;		
		if($cell.text()=="Poor") return 0;	
 }

For example, this is the html for the above table:

<table>
    <tr>
      <td class="dataType_alpha">Ice Cream Flavor </td>
      <td class="dataType_rating">Rating</td>
  </tr>
... (table continues)

Example 6 - Set Up Secondary Sort via CSS

When a user clicks on the flavor heading, the table will be sorted by flavor and then rating (ascending).

Ice Cream Flavor Rating
Vanilla Good
Chocolate Average
Strawberry Poor
Rocky Road Average
Rum Raisin Poor
Mint Chocolate Chip Good

In many instances, you want to do a multi-column sort. For example, when user clicks on the "Rating" heading, the rows are sorted first by rating and then by flavor. Flavor is always sorted ascending. You can set a "secondarySort" class to the header cell.

The secondary sort class is always in this format:

  • secondarySort_index_direction_index_direction (etc)

You can have as many secondary columns as needed. Index is the index of the header cell (or it's position from the left). The left most column has an index of 0.

Direction is always required. Direction must be one of the following values:

  • 1 for ascending
  • -1 for descending
  • 0 to have the same sort direction as the primary column (flip between ascending and descending).

For example, this is the html for the above table:

<table>
    <tr>
      <td>Ice Cream Flavor </td>
      <td class="dataType_rating  secondarySort_0_1">Rating</td>
  </tr>

1. Sort Class (Always Ascending)

2. Sort Class (Switch between Ascending and Descending)

You can sort a table via javascript by calling the sortTableColumn method applied to the table.

  • $(mytable).sortTableColumn(index,direction)

Where index is the index of the column to be sorted (the left-most column has an index of 0) and direction is the direction of the sort. Direction must be one of the following values:

  • 1 for ascending
  • -1 for descending
  • 0 flip between ascending and descending

For example, this is the code for the two links above:

<a href="javascript:void(0)" onclick="$('#classTable1').sortTableColumn(0,1)">1. Sort Class (Always Ascending)</a>

<a href="javascript:void(0)" onclick="$('#classTable1').sortTableColumn(0,0)">2. Sort Class (Switch between Ascending and Descending)</a>

You can also use the sortTableColumn to do multi column sorting. To do so, call tthe sortTableColumn this way:

  • $(mytable).sortTableColumn([[index,direction],[index,direction]])

You can set as many columns as needed.Index is the index of the column (the left-most column has an index of 0) and direction is the direction of the sort. Direction must be one of the following values:

  • 1 for ascending
  • -1 for descending
  • 0 flip between ascending and descending

For example, this is the code for the link above:

<a href="javascript:void(0)" onclick="$('#classTable2').sortTableColumn([[1],[0,1]])">1. Sort By Date and then by class ascending</a>

Althought is will probably only rarely be necessary, you can set a column to do secondary sort by using javascript. There are two methods to do this: "setSecondarySort" and "setSecondarySortByIndex". Both are applied to the header cell you want to add the secondary sort to and both apply the css to the header cell described in example 6.

The setSecondarySort method is in this format:

  • $(primaryTableCell).setSecondarySort(secondaryCellObj,direction,secondaryCellObj,direction);

Where the secondaryCellObj is a jQuery pointer to the secondary column cell header.

The setSecondarySortByIndex method is in this format:

  • $(primaryTableCell).setSecondarySortByIndex(index,direction,index,direction);

Where the index is the index of the secondary sort column.

 

For example, this is the html of the table above:

<table>
	<tr>
      <td id="classTable3-ClassColumn">Class</td>
      <td id="classTable3-DateColumn">Date Offered (month)</td>
    </tr>
